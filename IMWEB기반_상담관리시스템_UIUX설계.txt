# IMWEB 기반 VisionMakers 상담 관리 시스템 UI/UX 설계서

## 1. IMWEB 관리자 UI 분석 및 벤치마킹

### 1.1 IMWEB 관리자 페이지 핵심 특징
```
레이아웃 구조:
- 상단 헤더: 사이트명, 디자인 모드, 빠른 접근 링크
- 좌측 사이드바: 계층적 메뉴 구조
- 메인 콘텐츠: 대시보드 및 기능별 관리 영역
- 우측 패널: 계정 정보 및 리소스 링크

시각적 특징:
- 깔끔하고 미니멀한 인터페이스
- 중성적 색상 팔레트 (회색, 파란색 계열)
- 명확한 시각적 계층 구조
- 가독성 중심의 타이포그래피

기능적 특징:
- 직관적인 네비게이션
- 빠른 접근성 (Quick Access)
- 모바일 대응 (제한적)
- 통합 대시보드 중심
```

### 1.2 벤치마킹 포인트
```
✅ 적용할 요소:
1. 계층적 사이드바 네비게이션
2. 상단 컨텍스트 바 (사이트 정보 + 빠른 실행)
3. 카드 기반 대시보드 레이아웃
4. 중성적이면서 전문적인 색상 체계
5. 명확한 정보 그룹핑

🔄 개선할 요소:
1. 더 풍부한 데이터 시각화
2. 실시간 업데이트 표시
3. 고급 필터링 인터페이스
4. 모바일 최적화 강화
5. 접근성 개선
```
