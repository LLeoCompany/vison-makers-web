## 4. 섹션별 디자인 정의

### 4.1 Hero Section (히어로)
**목적**: 첫인상, 핵심 가치 전달, CTA 유도

**배경 요소**:
- 코드 에디터 UI (VS Code 스타일)
- 터미널 윈도우 (빌드 성공 메시지)
- 플로팅 코드 스니펫
- 연결된 노드/네트워크 그래픽
- 부드러운 블루 그라디언트 오버레이

**플로팅 요소**:
- 왼쪽 상단: 코드 스니펫 카드 (const project = { success: true })
- 오른쪽 상단: React 컴포넌트 태그 (<Component />)
- 왼쪽 하단: 터미널 (npm run build → Build successful)
- 오른쪽 하단: 성공 아이콘 세트

**핵심 메시지**:
- 배지: "500+ 프로젝트 완료"
- 헤드라인: "초기 구축부터 유지보수까지 완벽한 IT 파트너"
- 서브: "검증된 전문가들이 기획, 디자인, 개발을 한번에 해결합니다"

**통계 섹션**:
- 500+ 프로젝트 완료
- 98% 고객 만족도
- 24h 빠른 응대

### 4.2 Services Section (서비스)
**목적**: 제공 서비스 명확히 전달

**배경 요소**:
- 기획서/와이어프레임 아이콘
- 피그마/스케치 디자인 툴 아이콘
- 코드 브래킷 및 프로그래밍 언어 로고
- 서버/클라우드 인프라 아이콘
- 연결선으로 통합된 느낌

**서비스 카드**:
1. **전략 기획** 🎯
   - 비즈니스 목표 맞춤 전략
   - 사용자 리서치, 정보 구조 설계

2. **UI/UX 디자인** 🎨
   - 브랜드 아이덴티티 반영
   - 전환율 최적화 UX

3. **웹/앱 개발** 💻
   - 최신 기술 스택
   - 반응형 웹, 모바일 앱, 관리자 시스템

4. **유지보수** 🛡️
   - 6개월 무료 지원
   - 24시간 모니터링

## 5. 컴포넌트 스타일

### 버튼
```css
Primary: 블루 그라디언트 + 그림자
Secondary: 흰색 + 테두리
Ghost: 투명 + 호버시 배경
```

### 카드
```css
배경: 흰색
테두리: 1px #f1f5f9
그림자: 부드러운 그림자
호버: 살짝 위로 + 그림자 강화
모서리: 24px (2xl)
```

### 아이콘
- 크기: 64x64px (피처 아이콘)
- 스타일: 그라디언트 배경 또는 라이트 배경
- 이모지 사용 가능

## 7. 애니메이션

### 플로팅 요소
```css
float: 6s ease-in-out infinite (위아래 20px)
floatSlow: 8s ease-in-out infinite (위아래 15px + 회전)
pulse: 3s infinite (투명도 변화)
```

### 인터랙션
```css
호버: translateY(-4px) + 그림자 강화
전환: 200ms cubic-bezier(0.4, 0, 0.2, 1)
```
