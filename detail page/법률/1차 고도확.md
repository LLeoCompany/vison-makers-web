# 법률 상세 페이지(Legal) 디자인 및 콘텐츠 고도화 요청

현재 데모 페이지를 아래 가이드라인에 맞춰 '법률 전문성'이 느껴지도록 리뉴얼해줘.

1. 색감 변경 (Legal Specialization):
   - 메인 페이지의 밝은 블루 대신, 더 어둡고 묵직한 'Midnight Navy (#1A2238)'를 메인으로 사용해줘.
   - 배경은 Pure White보다는 아주 연한 'Ivory Grey (#F8FAFC)'를 사용하여 고급 법률 문서의 느낌을 줘.
   - 포인트 컬러(중요 버튼이나 강조 아이콘)에는 'Antique Gold (#B89150)'를 사용하여 권위를 더해줘.

2. 메인 배너 업그레이드 (Hero Section):
   - 배경에 '법전과 서류가 디지털 데이터 노드로 변환되는 영상'을 슬라이드 형태로 배치해줘.
   - 배너 텍스트에 "법률 지식과 AI의 결합, 단순 검색을 넘어 추론의 영역으로"라는 메시지를 강조해줘.

3. 상세 설명 보강 (Content Detail):
   - '왜 법률에 RAG가 필요한가?' 섹션을 추가해줘.
   - 단순 GPT의 환각 현상을 법률 전문가들이 왜 경계해야 하는지 설명하고, Vision AI의 '근거 기반 답변(Source Citation)' 기능을 상세히 시각화해줘.
   - 주요 기능 리스트: 판례 교차 검색, 계약서 독소 조항 자동 탐지, 비정형 법률 문서의 정형화 분석.

4. UI/UX 디테일:
   - 답변 옆에 사건번호와 판결문 원문이 뜨는 '법률 전용 챗 UI' 예시 컴포넌트를 페이지 중간에 배치해줘.
   - 모바일에서도 법조문을 읽기 편하도록 자간과 행간을 넓게 조정해줘.

"이 수정을 통해 법무법인 파트너들이 보기에 '가장 신뢰할 수 있는 법률 도구'처럼 보이게 만들어줘."
