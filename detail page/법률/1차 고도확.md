# 법률 상세 페이지(Legal) 3차 고도화: 시각적 대비 및 인터랙션 강화

현재의 너무 하얀 디자인을 탈피하고, 법률 전문가에게 신뢰를 줄 수 있는 무게감 있는 페이지로 리뉴얼해줘.

1. 히어로 배너 (Hero Section) 개선:
   - 배경을 단순 화이트가 아닌, '다크 네이비 오버레이가 적용된 전문적인 법률 영상이나 그래픽'으로 교체해줘.
   - 텍스트는 흰색(#FFFFFF)으로 적용하여 가독성을 높이고, "법률 지식과 AI의 결합"이 시각적으로 느껴지도록 '법전'과 '데이터 노드'가 연결되는 비주얼을 상단에 배치해줘.

2. 콘텐츠 가독성 및 대비 (Contrast) 강화:
   - 모든 섹션이 하얗게 보이지 않도록 'Zebra Striping' 기법을 써줘. (섹션 1: 화이트 / 섹션 2: 연한 그레이 #F1F5F9 / 섹션 3: 다크 네이비 #0F172A)
   - 카드 컴포넌트에 은은한 그림자(shadow-lg)와 아주 얇은 블루 테두리(border-blue-100)를 추가해서 배경과 분리시켜줘.

3. 하단 검색창/입력창 엔터 이벤트 구현:
   - 페이지 하단에 "궁금하신 법률 AI 기능을 입력해보세요"라는 큰 입력창을 만들어줘.
   - 사용자가 텍스트를 입력하고 'Enter'를 누르면, 입력한 내용이 상담 신청 위젯의 '문의 내용'으로 자동 전달되면서 상담 위젯(Sheet)이 부드럽게 열리게 해줘. (onKeyDown 이벤트 처리)

4. 기술적 포인트:
   - 엔터 입력 시 "분석 중..."이라는 짧은 애니메이션을 보여준 뒤 상담 폼으로 넘어가면 더 전문적으로 보일 것 같아.
   - 법률 특유의 묵직한 서체 느낌을 위해 헤드라인의 자간(letter-spacing)을 좁히고 굵게 처리해줘.
