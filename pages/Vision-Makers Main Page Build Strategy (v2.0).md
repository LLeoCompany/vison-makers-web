   규격 최적화: 모든 섹션은 브라우저 높이(100vh)에 맞춘 Snap Scroll(스크롤 섹션 고정) 방식을 권장합니다.

정보 다이어트: 클라이언트가 쉽게 이탈하지 않도록 상세 수치보다는 직관적인 비주얼 키워드 중심으로 구성합니다.

아이덴티티: '보안 솔루션' 느낌보다 **'일 잘하는 로컬 AI 비서'**가 우리 서버 안에 살고 있다는 느낌을 줍니다.

2. 섹션별 상세 기획
   [Section 1: Hero - 명확한 정체성]
   배경: 복잡한 그래픽 제거. 심플한 다크 네이비 배경에 중심부에 집중되는 은은한 빛(Spotlight) 효과.

메인 카피: "우리 서버 안에 직접 심는, 가장 똑똑한 AI 사내 비서."

서브 카피: "데이터 유출 걱정 없는 로컬 인프라 기반 맞춤형 AI 이식."

CTA: [비서 도입 상담] (강렬한 블루), [구축 사례] (투명 고스트 버튼).

디자인 근거: 불필요한 수식어를 덜어내고 '로컬 AI'와 '사내 비서'라는 본질에 집중합니다.

[Section 2: Core Value - 로컬 AI의 강점]
비주얼: 섹션을 3등분하여 카드 형태로 배치하되, 텍스트는 최소화.

내용:

보안: "성벽 안의 지능" - 데이터 유출 0건.

성능: "압도적 속도" - 로컬 망 내 실시간 처리.

정확도: "우리 회사 맞춤" - 92% 이상의 정밀한 답변.

UX: 스크롤 시 카드가 중앙으로 모이며 강조되는 애니메이션.

[Section 3: Industry Solutions - 업종별 비서 (한눈에 보기)]
기존 문제 해결: 기존의 긴 리스트 방식 대신, 가로 슬라이드 혹은 아이콘 그리드로 변경하여 한 화면에 모든 업종이 들어오게 합니다.

항목:

⚖️ 법률: 판례 검색 및 리스크 검토.

🩺 의료: 진단 보조 및 환자 상담 자동화.

🏭 제조: 매뉴얼 검색 및 불량 원인 분석.

🚀 스타트업: 지식 베이스 구축 및 온보딩.

UX: 아이콘 클릭 시 해당 업종의 핵심 기능 1줄만 간결하게 노출.

[Section 4: Security Architecture - 직관적 도식]
텍스트 가감: 기술적인 설명문 전체 삭제.

비주얼: **[사내 문서] → [로컬 AI 서버] → [임직원]**으로 이어지는 직선형 흐름도만 크게 배치.

강조 문구: "인터넷 연결 없이도 작동하는 완벽한 폐쇄형 지능."

[Section 5: Footer - 재구성]
기존 문제 해결: 너무 넓게 퍼진 정보를 하단부 중앙으로 밀집시켜 안정감을 줍니다.
