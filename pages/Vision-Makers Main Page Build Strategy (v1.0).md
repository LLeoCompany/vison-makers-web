📑 Vision-Makers 메인 페이지 구축 가이드 (v1.0)

1. 프로젝트 개요
   프로젝트명: Vision-Makers 브랜드 홈

핵심 정체성: 기업 전용 Private RAG AI 이식 파트너

디자인 컨셉: 하이엔드 다크 모드, 사이버 보안, 일렉트릭 블루 액센트

목표: '보안 진단' 유도 및 '업종별 데모' 페이지로의 허브 역할

2. 권장 기술 스택
   Framework: Next.js (App Router)

Styling: Tailwind CSS

Animation: Framer Motion (데이터 흐름 및 보안 시각화 필수)

Icons: Lucide React (심플하고 전문적인 아이콘 세트)

3. 섹션별 상세 기획 및 UX 구성
   [Section 1: Hero - 정체성 선언]
   배경: 딥 블랙(#0a0a0f) 그라데이션, 미세하게 움직이는 그리드 패턴.

메인 카피(H1): "외부 AI는 유출되지만, Vision-Makers는 이식합니다."

'이식합니다' 부분에 블루 그라데이션 및 글로우 효과 적용.

서브 카피: "사내 문서를 성벽 안에 가두고, 오직 지능만 활용하는 우리 회사 전용 Private RAG 솔루션."

CTA 버튼: - 버튼 A(기본): "무료 보안 리스크 진단 🔍" (블루 채우기)

버튼 B(보조): "업종별 데모 보기" (아웃라인 스타일)

[Section 2: Social Proof - 신뢰의 지표]
디자인: 가로형 바 형태, 3가지 핵심 수치 강조.

수치 항목:

100,000+ | DevGym 실증 유저 (시스템 안정성)

92% | 답변 정확도 (RAG 최적화 기술력)

ZERO | 데이터 외부 유출 사고 (철저한 보안성)

UX: 스크롤 시 숫자가 0부터 올라가는 카운팅 애니메이션 적용.

[Section 3: 보안 아키텍처 - '성벽(The Vault)' 다이어그램]
컨셉: 데이터가 내부망을 벗어나지 않음을 시각적으로 증명.

레이아웃: 좌(수집) -> 중(성벽 안 저장/처리) -> 우(필터링 후 질의) 흐름.

시각 요소:

입력: 문서 아이콘들이 자물쇠(🔐) 아이콘을 통과.

중앙: 거대한 방패(🛡️) 테두리 안에 'Vector DB'가 펄스 애니메이션으로 작동.

출력: 외부망으로 나갈 때는 개인정보가 비식별화(Masking)되는 필터 레이어 시각화.

캡션: "AES-256 암호화", "전용 VPC 독립 구축", "실시간 PII 필터링".

[Section 4: 업종별 솔루션 허브 - 카드형 그리드]
디자인: 3x3 혹은 4x2 그리드 레이아웃.

구성: 아이콘 + 업종명 + 핵심 한 줄 솔루션.

목록: 법률, 의료, 제조, 스타트업, 광고, 공공, 프랜차이즈 등.

UX: 마우스 오버 시 카드가 블루 톤으로 발광하며 '상세 보기 →' 링크 노출.

[Section 5: 비교 섹션 - 외부 LLM vs Vision-Makers]
디자인: 대비되는 2분할 레이아웃 혹은 비교 테이블.

내용:

일반 AI: 데이터 학습 활용 위험, 외부 서버 전송, 환각 현상 유발.

Vision-Makers: 사내 서버 독립 구축, 학습 원천 차단, 근거(출처) 기반 답변.

[Section 6: 하단 CTA - 최종 전환]
디자인: 유리 질감(Glassmorphism) 효과가 적용된 대형 카드.

카피: "이미 준비된 귀사의 데이터를 안전한 지능으로 바꿀 차례입니다."

입력폼: [이메일 주소 입력란] + [보안 진단 요청 버튼].

AI 코딩을 위한 스타일 가이드 (Prompt)

1. **컬러 팔레트:**
   - 배경: #0a0a0f (Deep Black)
   - 포인트: #00D1FF (Electric Blue)
   - 강조: #60A5FA (Soft Blue)
   - 경고/위험: #EF4444 (외부 유출 리스크 표현용)

2. **폰트 및 타이포그래피:**
   - 프리텐다드(Pretendard) 혹은 Inter 사용.
   - 제목은 두껍게(Bold), 자간을 살짝 좁혀 전문적인 느낌 강조.

3. **애니메이션 원칙:**
   - Framer Motion의 'initial', 'whileInView'를 적극 활용할 것.
   - 모든 섹션은 스크롤 시 아래에서 위로 부드럽게 페이드인될 것.

4. **어조(Tone of Voice):**
   - 전문적이고 차분하며, 보안에 있어 타협하지 않는 단호한 톤.
