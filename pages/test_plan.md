## [Typography & Text Color Spec]

### 1. 전역 텍스트 규칙

- 기본 배경(#FFFFFF) 위에서 모든 화이트 글자 제거.
- 메인 타이틀은 `#172B4D`를 사용하여 묵직한 신뢰감 형성.
- 본문 텍스트는 `#42526E`를 사용하여 눈의 피로도 감소.

### 2. 섹션별 텍스트 적용

- **Hero Section:** - "당신의 5시간" -> Deep Navy (#172B4D)
  - "5초면 충분합니다" -> Royal Blue (#0052CC) 강조
- **Card Section:** - 카드 제목은 볼드 처리된 Deep Navy.
  - 카드 설명은 Steel Gray.
- **Popup/Modal:**
  - 팝업 내 모든 텍스트는 화이트 배경에 맞춘 다크 컬러 시스템 적용.
  - 단, 버튼 내부나 강조용 배지 내부에서만 화이트 글자 허용.

### 3. 대비(Contrast) 점검

- WCAG 2.1 가이드라인에 따라 텍스트와 배경의 대비비를 최소 4.5:1 이상 유지할 것.
