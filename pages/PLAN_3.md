# 💎 Vision-Makers v11.0: Customer-Centric Conversion Funnel

**"검증된 엔진을 당신의 비즈니스에 이식하여 실질적인 ROI를 만듭니다."**

## 1. 메인 슬로건 및 히어로 전략 (Hero & Message)

- **핵심 메시지:** "검증된 RAG 기술을 **귀사에 맞게 구축합니다.**"
- **디자인 포인트:** - '귀사'라는 단어에 `Electric Cyan` 글로우 효과와 그래디언트 적용.
  - 배경에 데이터가 유기적으로 연결되는 파티클 애니메이션을 배치하여 '이식(Integration)'의 느낌 강조.

---

## 2. 서비스별 직관적 내비게이션 (Service-Specific Funnel)

고객이 고민(AI 구축, 챗봇, 추천)에 따라 즉시 해답을 찾고 상담으로 이어지는 구조입니다.

| 서비스 구분             | 클릭 시 노출되는 핵심 가치 (Drawer/Section)                    | 상담 전환 텍스트 (Micro-copy)    |
| :---------------------- | :------------------------------------------------------------- | :------------------------------- |
| **01. RAG 시스템 구축** | **[보안/정확도]** 사내 지식 베이스 구축 사례 & 정확도 92% 인증 | "사내 데이터 AI 자산화 진단받기" |
| **02. AI 챗봇 개발**    | **[효율/자동화]** DevGym CS 50% 절감 사례 & 1.2초 즉시 응답    | "인건비 절감 시뮬레이션 신청"    |
| **03. AI 추천 시스템**  | **[매출/ROI]** 유저 행동 기반 매출 3배 향상 사례 & 이탈률 감소 | "매출 향상 엔진 설계 문의"       |

## 3. 섹션별 상세 기획 (Section Details)

### [Main Section] 검증된 RAG 솔루션 (3-Bento Card)

- **UX 전략:** 3개의 카드를 '메뉴판'처럼 직관적으로 구성.
- **디테일:** 각 카드 하단에 성과 지표 배지(`ROI 3배`, `자동화 78%`, `보안 99.9%`)를 노출하여 클릭 유도.
- **인터랙션:** 카드 클릭 시 페이지 이동 없이 **측면 드로워(Drawer)**가 열리며 '사례 → 플로우 → 상담 폼'이 원스톱으로 등장.

### [Drawer Contents] 서비스별 상세 퍼널 (Deep-Dive)

1. **Case Study:** 해당 서비스가 적용된 실제 DevGym 대시보드 스크린샷 노출.
2. **Success Flow:** "귀사의 데이터 수집 → 엔진 커스텀 → 실전 배포" 3단계 시각화.
3. **Quick Contact:** 해당 서비스 전용 상담 폼 (예: "챗봇 도입 시 예상 절감 비용을 알려드립니다").

---

## 4. 디자인 가이드: 프리미엄 & 직관성 (Visual Identity)

- **Typography:** 메인 카피는 `Pretendard Bold`, 기술 수치와 주석은 `JetBrains Mono`를 혼용하여 전문성 강화.
- **Color Logic:** 각 서비스마다 고유의 악센트 컬러 부여 (챗봇-Blue, 추천-Cyan, 구축-Indigo) 하여 사용자 인지 속도 향상.
- **Layout:** "싸 보이는" 느낌을 지우기 위해 섹션 간 여백을 충분히 확보하고, 미세한 노이즈 텍스처 배경 적용.

---

## 5. 바이브 코딩용 최종 지시문 (Master Prompt)

> **"v11.0 기획안에 맞춰 '귀사 맞춤형 구축'과 '서비스별 전환 퍼널'을 구현해줘."**
>
> 1. **메인 메시지 부각:** "검증된 RAG 기술을 귀사에 맞게 구축합니다"를 히어로 섹션 바로 아래에 가장 강조해서 배치해줘.
> 2. **서비스 클릭 인터랙션:** 'RAG 구축', 'AI 챗봇', '추천시스템' 3개 카드 중 하나를 클릭하면 즉시 해당 서비스의 상세 사례와 구축 플로우가 담긴 **우측 드로워(Drawer)**가 나타나게 해줘.
> 3. **드로워 내부 퍼널:** 드로워 안에는 [실제 성공 사례 이미지] -> [3단계 구축 과정 아이콘] -> [전용 상담 신청 폼]이 순차적으로 배치되어야 해.
> 4. **ROI 가시화:** 모든 수치(ROI 3배, 92% 등)에 탄성(Spring) 애니메이션을 적용하고, '귀사'라는 단어에 Cyan 컬러 글로우 효과를 줘.
> 5. **모바일 최적화:** 모바일에서는 3개 서비스를 탭(Tab) 형태로 배치하고, 하단에 항상 고정된 '맞춤형 진단 신청' Sticky 버튼을 만들어줘.
> 6. **디자인 마감:** 전체적으로 다크 모드의 깊이감을 살리고, 기술 로고들을 Grayscale로 나열하여 엔터프라이즈 급의 신뢰도를 완성해줘.

---

## 6. 기대 효과 (Goal)

- **직관성:** 고객이 본인의 문제를 해결할 카드를 찾는 데 걸리는 시간 3초 이내.
- **신뢰도:** "귀사에 맞게"라는 카피와 실제 사례 결합으로 외주 플랫폼 이상의 파트너십 형성.
- **전환율:** 서비스 탐색부터 상담 신청까지의 동선을 단축하여 **이탈률 30% 감소 및 상담 리드 2배 증가**.
