# 🚀 Project: Vision-Makers RAG Platform Final v7.5

**"성과 시각화(Performance) + 실전 증명(Evidence) + 심리적 전환(UX)"**

## 1. Design System: Impact & Trust

- **Base Color:** `#0A0A0F` (Deep Space Black) - 전문성
- **Action Color:** `#E94560` (Vivid Crimson) - 행동 유도
- **Success Color:** `#48BB78` (DevGym Green) - 성과 및 보안 완료
- **Point Color:** `#00BFFF` (Electric Cyan) - 기술적 지능
- **Interaction:** Glassmorphism, Spring Animation, Bento Grid

---

## 2. 핵심 섹션별 규합 가이드 (The Winning Flow)

### [Section 1] Hero: Result First

- **배치:** 메인 카피 하단에 3개의 핵심 성과 배지(Badge) 배치.
  - `ROI 300%↑` | `10만+ 실전검증` | `정확도 92%`
- **데모:** 산업군 선택 탭(금융/의료/제조)에 따라 답변의 구체적 비즈니스 수치(이탈률, 매출 등)가 변동되는 가변형 데모.

### [Section 2] NEW: Comparison UI (The Logic)

- **목표:** 기존 방식 대비 Vision-Makers RAG의 압도적 효율성 증명.
- **UI:** 수평 바 차트(Horizontal Bar Chart) 시각화.
  - **응답 속도:** [기존] 120분 ---------------- vs [Vision] 1.2s (초단기 바)
  - **운영 비용:** [기존] 100% --------------- vs [Vision] 35% (절감)
- **효과:** 기술을 모르는 결정권자에게 도입의 ROI를 즉각 인식시킴.

### [Section 3] DevGym Proof: Achievement Bento Grid

- **배치:** 실제 운영 스크린샷 6장 (관리자/앱/추천 엔진).
- **디테일:** 각 이미지 우측 상단에 **'Achievement Tag'** 오버레이.
  - 예: `[Response Time 1.2s]`, `[Accuracy 92%]`.
- **효과:** 정적인 이미지를 실제 성과 지표와 결합하여 증거력 극대화.

### [Section 4] Security Layer: The Trust Shield

- **구성:** 4대 보안 약속 (데이터 격리, 비학습, 권한 제어, 암호화).
- **디자인:** 아이콘 중심의 인포그래픽으로 복잡한 보안을 한눈에 안심시킴.

### [Section 5] Conversion: Smart Audit Form

- **UX:** 폼 상단에 부드럽게 채워지는 **진행률 인디케이터** (예: "3/8 단계 완료").
- **완료:** 모든 항목 입력 시 버튼이 `#48BB78`로 변하며 **Pulse(심박)** 효과 발생.

## 3. 핵심 애니메이션 가이드 (The Vibe)

1. **Spring Count-up:** 모든 수치가 올라갈 때 목표값에서 살짝 튕겼다 돌아오는 탄성 효과 적용.
2. **Skeleton Loading:** 데모 질문 입력 후 답변 출력 전 0.8초간 '생각하는' 시각적 로딩 제공.
3. **Mobile Sticky CTA:** 모바일 하단에 `#E94560` 기반의 Glassmorphism 고정 버튼 상시 노출.
