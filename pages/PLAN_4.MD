# 🚀 Vision-Makers v12.0: Hero-Funnel Master Build

**"검증된 기술을 메인 베너로 격상하고, 클릭 한 번에 [사례-기술-상담]을 연결한다"**

## 1. 구조 개편 (Section Overhaul)

- **As-Is:** 기존 Hero 섹션 + 서비스 소개 섹션 분리.
- **To-Be:** 기존 Hero를 삭제하고 **[검증된 RAG 기술... 구축합니다]** 섹션을 **최상단 메인 배너(Hero)**로 격상.
- **Goal:** 접속 즉시 '우리의 전문성'과 '고객의 선택지'를 한 화면에 노출.

---

## 2. 메인 배너(New Hero) 상세 가이드

### [Visual & Copy]

- **Main Heading:** "검증된 **RAG 기술**을 귀사에 맞게 **구축**합니다"
- **Styling:** '귀사'와 '구축' 키워드에 `Electric Cyan(#00BFFF)` Gradient 및 네온 Glow 효과 적용.
- **Background:** `Deep Space Black(#0A0A0F)` 배경에 미세한 격자(Grid) 패턴과 유기적인 데이터 흐름 애니메이션 적용.

### [3-Action Cards]

- **Card UI:** 3분할 벤토 그리드 형태. 호버 시 `scale-105` 및 `Border Glow` 효과.
- **Card List:**
  1. **RAG 시스템 구축:** [데이터 보안 99.9% / 정확도 92%]
  2. **AI 챗봇 개발:** [인건비 50%↓ / 1.2s 응답] (Badge: Most Popular)
  3. **추천 시스템:** [ROI 300%↑ / 유저 유지율 강화]
- **Interaction:** 카드 클릭 시 우측에서 슬라이드되는 **전환 드로워(Drawer)** 호출.

---

## 3. 서비스별 전환 퍼널 (Drawer Contents)

고객이 카드를 클릭했을 때 나타나는 '설득의 여정'입니다.

- **Step 1. 성공 사례 (Evidence):** - 해당 서비스가 적용된 실제 DevGym 대시보드 스크린샷 노출.
  - `[Accuracy 92%]`, `[CS Cost -50%]` 등 성과 지표 태그 오버레이.
- **Step 2. 구축 플로우 (Process):** - "귀사 데이터 분석 → 맞춤형 엔진 튜닝 → 보안 배포" 3단계 인포그래픽.
- **Step 3. 기술력 소개 (Tech Specs):** - 텍스트 벽 대신 'JetBrains Mono' 폰트를 활용한 핵심 기술 스펙 3~4줄 요약.
- **Step 4. 상담 신청 (Action):** - "귀사에 맞는 [서비스명] 구축 진단받기" 버튼 및 간이 상담 폼.

---

## 4. 디자인 및 UX 마감 (High-End Polish)

- **여백:** 섹션 간 간격을 대폭 늘려(Py-32 이상) 프리미엄 솔루션 느낌 구현.
- **폰트:** 헤드라인(Pretendard Bold), 수치/기술데이터(JetBrains Mono).
- **애니메이션:** - 모든 카운트업 수치에 **Spring 효과** 적용.
  - 드로워 오픈 시 배경에 `Backdrop-blur(10px)` 적용하여 깊이감 형성.
- **모바일:** 하단 고정형 **Glassmorphism CTA** ("ROI 3배 무료 진단 신청") 상시 노출.

---

## 6. 기대 효과

- **전환율:** 탐색 과정을 생략하고 메인에서 즉시 퍼널로 진입시켜 상담 전환율 1.5배 상승.
- **전문성:** 사례와 기술력을 하나의 드로워에서 보여줌으로써 '외주'가 아닌 '솔루션 파트너' 이미지 구축.
- **가독성:** 훑어보기(Home)와 상세검증(Drawer)의 완벽한 분리로 사용자 피로도 최소화.
