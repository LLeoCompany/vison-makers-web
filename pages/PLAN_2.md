# 🚀 RAG-LLM Agency Master Plan v6.0 (Optimized for Conversion)

**"10만 유저 검증(DevGym) + 산업별 맞춤(Custom) + 엔터프라이즈 보안(Security)"**

## 1. 디자인 시스템 & 컬러 심리학 (The Visual Vibe)

- **Base:** `#0A0A0F` (Deep Space Black) - 기술적 권위와 묵직한 신뢰감.
- **Action (CTA):** `#E94560` (Vivid Crimson) - 시선 강탈 및 즉각적 행동 유도.
- **Success/Safe:** `#48BB78` (DevGym Green) - 실전 검증 완료 및 보안 안전 신호.
- **Tech/Logic:** `#00BFFF` (Electric Cyan) - 지능형 RAG 파이프라인의 명쾌함.

---

## 2. 섹션별 고도화 가이드 (Client-Centric UX)

### [Section 1] Hero: Industry-Specific RAG Demo

- **핵심:** "내 데이터도 될까?"라는 의문을 즉시 해소.
- **기능:** [계약서 | 로그 | EMR | 상담기록] 선택 탭(Tab) 또는 드롭다운 배치.
- **모션:** 선택 즉시 0.8초간 **스켈레톤 로딩(Skeleton Loading)** 노출 후 답변 출력.
- **배경:** 선택한 산업 컬러에 맞춘 파티클이 시속 20px로 부유하는 Canvas 효과.

### [Section 2] Service Cards: ROI & Popularity

- **디테일:** 모든 아이콘 48x48px 고정. 중앙 카드에 `Glow` 및 `Gradient Border` 적용.
- **인터랙션:** Hover 시 `scale-110` 및 `#E94560` 글로우. 'Most Popular' 뱃지는 상하로 부드럽게 Bounce.

### [Section 3] DevGym Proof: The Real Evidence (Critical)

- **업데이트:** 플레이스홀더 삭제 → **실제 관리자/앱 스크린샷 6장**을 벤토 그리드로 배치.
- **UX:** 이미지 호버 시 "피트니스 이탈 예측" → "의료 기록 자동화"로 텍스트가 부드럽게 모핑(Morphing).

### [Section 4] Security Layer: Trust Shield (New)

- **메시지:** "AI는 위험하지 않습니다. 기존 보안보다 더 안전합니다."
- **4대 핵심 가치 (아이콘 중심):**
  1. **독립 공간:** 고객별 벡터 DB 완전 격리.
  2. **비학습 원칙:** 외부 LLM 학습에 데이터 사용 차단.
  3. **권한 제어:** RBAC 기반 문서 접근 필터링.
  4. **철저 암호화:** 원문 데이터 AES-256 저장.

### [Section 5] Conversion: Smart Audit Form

- **UX:** 폼 상단에 **'진행률 인디케이터'** (예: "3/8 단계 완료") 배치하여 완수 심리 자극.
- **완료 효과:** 모든 필수 항목 입력 시 제출 버튼이 `#48BB78` 그린으로 변하며 **Pulse(심박)** 애니메이션 작동.

## 3. 바이브 코딩용 최종 지시문 (Vibe Coding Prompt)

> **"Next.js와 Tailwind로 구축된 현 사이트를 다음 v6.0 기획에 맞춰 정밀 고도화해줘."**
>
> 1. **가변형 데모:** Hero 섹션에 '산업군 선택' 탭을 만들고, 선택에 따라 질문과 답변 내용이 스켈레톤 로딩과 함께 바뀌는 로직을 구현해줘.
> 2. **실전 이미지:** 섹션 3의 이미지를 실제 준비된 DevGym 스크린샷으로 교체하고, 호버 시 타 산업으로의 확장성을 보여주는 텍스트 애니메이션을 넣어줘.
> 3. **숫자 애니메이션:** 모든 수치(Metric)에 뷰포트 진입 시 2초간 카운트업(Count-up) 효과를 적용해줘.
> 4. **보안 인포그래픽:** '엔터프라이즈 보안 레이어' 섹션을 신설하고, 4가지 보안 약속을 아이콘 위주로 깔끔하게 시각화해줘.
> 5. **모바일 최적화:** 375px 모바일 환경에서 Hero 데모 높이를 280px로 고정하고, 하단에 항상 고정된 '상담 신청' 버튼(Sticky CTA)을 생성해줘.
> 6. **폼 인터랙션:** 상담 폼에 진행바를 추가하고, 입력 완료 시 버튼에 그린 펄스 효과를 줘서 행동을 유도해줘.

## 4. 데이터 분석 설계 (Analytics)

- **GA4:** `demo_tab_click`, `form_progress_step`, `security_view` 이벤트 트래킹.
- **Hotjar:** 상담 폼에서 사용자가 가장 오래 머무는 항목과 이탈 지점 분석용 히트맵 설정.
