# [기획서] Vision AI 플랫폼 서비스 고도화 및 UI/UX 상세 설계

## 1. 프로젝트 목표

- **사용자 경험 개선**: 직관적인 네비게이션과 인터랙티브한 요소 도입
- **신뢰도 강화**: 화이트/블루 테마와 영상 콘텐츠를 통한 전문성 전달
- **전환 최적화**: 즉각적인 상담 신청 위젯 및 분야별 상세 페이지 구성

## 2. 디자인 시스템

- **Main Color**: Trust Blue (#1E40AF), White (#FFFFFF)
- **Point Color**: Sky Blue (#3B82F6), Slate Grey (#64748B)
- **Layout**: Sticky Header, Full-width Hero, Grid-based System Info

## 3. 상세 구조 및 기능 (Structure)

### [1] 헤더 (Header)

- **위치**: 상단 고정 (Sticky)
- **구성**:
  1. **좌측**: Vision AI 로고
  2. **중앙 (메뉴)**:
     - 회사소개
     - 분야별 AI (Dropdown): 법률, 광고, 식품, 세무, 제조, 의료
     - LLM이란?
  3. **우측**: [상담 신청] 버튼 (클릭 시 우측 하단에서 상담 위젯 오픈)

### [2] 메인 배너 (Hero Section)

- **배경**: 고해상도 기술 관련 영상 재생 (Autoplay, Muted, Loop)
- **오버레이**: 반투명한 블루/다크 레이어를 입혀 텍스트 가독성 확보
- **텍스트**: "데이터의 가치를 지능으로, Vision AI RAG 솔루션"
- **CTA**: 영상 위로 [솔루션 바로가기] 버튼 배치

### [3] 시스템 소개 섹션 (System Intro)

- **위치**: 메인 배너 바로 아래
- **콘텐츠**:
  - **Vision RAG Core**: 하이브리드 검색 및 자가 수정 엔진 설명
  - **보안 아키텍처**: 5계층 방어 체계 및 온프레미스 배포 강점 간략 요약
  - **데이터 주권**: 기업 데이터 유출 없는 폐쇄형 LLM 강조

### [4] 분야별 AI 상세 페이지 (Solution Pages)

- **기능**: 헤더 드롭다운에서 각 분야 클릭 시 `/solutions/[field]` 경로로 이동
- **콘텐츠 구성**:
  - 해당 산업의 페인 포인트 (예: 법률 문서 검색의 어려움)
  - 도입 후 기대 효과 (수치 중심)
  - 실제 적용 사례 UI 미리보기

### [5] 상담 신청 위젯 (Consultation Widget)

- **동작**: 헤더 상단 버튼 클릭 시 슬라이드 인(Slide-in) 또는 팝업 형태로 오픈
- **구성**: 이름, 연락처, 회사명, 관심 분야 선택, 문의 내용 입력 칸

## 4. AI 개발 가이드 (Prompt)

"위 기획서를 바탕으로 Next.js 14 프레임워크와 Tailwind CSS를 사용하여 웹사이트를 개발해줘. 특히 다음 사항을 엄격히 준수해줘."

1. **헤더 드롭다운**: 헤더의 '분야별 AI' 메뉴에 마우스를 올리면 법률, 광고, 식품, 세무 등의 항목이 깔끔한 애니메이션과 함께 나타나야 함.
2. **비디오 배경**: 메인 배너에 `video` 태그를 사용하여 배경 영상이 끊김 없이 재생되도록 구현하고, 그 위에 화이트 텍스트가 잘 보이도록 배경 레이어를 조정해줘.
3. **상담 신청 위젯**: 상단 버튼 클릭 시 화면 우측에서 부드럽게 나타나는 상담 신청 사이드바(Sheet 컴포넌트)를 구현해줘.
4. **분야별 페이지 이동**: 각 분야 클릭 시 해당 섹션으로 스크롤되거나 별도 상세 페이지로 이동하는 라우팅 구조를 잡아줘.
5. **테마**: 전체적으로 화이트 배경에 신뢰감을 주는 블루 계열 포인트 컬러를 유지해줘.
