# [기획서] Vision AI(Vision-Makers) 플랫폼 리뉴얼 및 서비스 소개 페이지

## 1. 프로젝트 개요

- **목표**: 기존 블로그 형태를 탈피하고, 신뢰 기반의 엔터프라이즈 RAG 솔루션 플랫폼 구축
- **핵심 키워드**: 데이터 주권, 보안, 전문성, 즉각적 도입
- **디자인 테마**: White & Blue (신뢰와 청결함 강조)

## 2. 디자인 시스템 (Design System)

- **Base Color**: Pure White (#FFFFFF) - 전체 배경 및 카드 배경
- **Primary Color**: Trust Blue (#1E40AF) - 헤드라인, 주요 버튼(CTA), 신뢰 포인트 아이콘
- **Accent Color**: Sky Blue (#3B82F6) - 강조 텍스트, 호버 효과
- **Sub Color**: Light Grey (#F8FAFC) - 섹션 구분 및 배경 대비용
- **Typography**: Pretendard 또는 Sans-serif 계열 (굵기 대비를 통한 정보 계층화)
- **Layout**:
  - 포스팅 리스트 형식을 완전히 제거
  - **Bento Grid** 및 **Layered Card** 스타일을 활용한 솔루션 중심 레이아웃

## 3. 페이지 구성 및 콘텐츠 (Sitemap)

### [Section 1] Hero Section (임팩트)

- **메시지**: "귀사의 데이터를 강력한 지능으로. 보안 기반 도메인 특화 RAG 솔루션"
- **Visual**: 화이트 배경에 블루 포인트가 들어간 추상적인 데이터 네트워크 그래픽
- **CTA**: [무료 도입 진단 시작하기] (Blue 버튼), [기술 스펙 보기] (Outline 버튼)

### [Section 2] RAG Solution Types (서비스 소개)

_3가지 타입의 RAG 서비스를 카드 형태로 제시 (Bento Grid 권장)_

1. **Type A: 사내 지식 기반 RAG (Internal Hub)**
   - 설명: "흩어진 사내 매뉴얼과 문서를 통합 학습."
   - 핵심 가치: 인수인계 효율화, 사내 보안 규정 완벽 준수.
2. **Type B: 전문 도메인 특화 RAG (Professional Expert)**
   - 설명: "법률, 금융, 제조 공정 등 고도의 전문 문서 분석."
   - 핵심 가치: 정확한 출처(Source) 제시, 전문가 수준의 추론 능력.
3. **Type C: 고객 접점 상담 RAG (Customer Specialist)**
   - 설명: "실시간 DB 연동을 통한 정확한 고객 응대."
   - 핵심 가치: 24/7 무중단 대응, 브랜드 맞춤형 톤앤매너 적용.

### [Section 3] Trust & Security (신뢰 지표)

- 기업 고객이 안심할 수 있는 '5-Layer 보안 체계' 시각화
- ISO 27001, On-Premise 구축 가능성, 데이터 유출 방지 기술 강조

### [Section 4] Immediate Consultation (상담 신청 플로우)

_단순 이메일 기재가 아닌 단계별 입력 폼_

- **Step 1**: 도입 목적 선택 (사내 지식 / 전문 분야 / 고객 응대)
- **Step 2**: 보유 데이터 규모 및 환경 (Cloud / On-Premise) 선택
- **Step 3**: 기업 정보 입력 및 미팅 예약 (Calendly 스타일 UI)
